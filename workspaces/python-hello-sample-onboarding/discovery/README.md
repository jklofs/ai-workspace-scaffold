# Discovery

Discovery documents help understand the problem before writing requirements.

Use this directory for research notes, stakeholder analysis, competing interpretations, open questions, assumptions, risks, and cross-workspace impact.

Create discovery documents with `./scripts/scaffold.sh create-doc <workspace-name> discovery <topic-slug>`. Use [../../../templates/documents/discovery.md](../../../templates/documents/discovery.md) manually only when the CLI cannot express the specialized file.
