# Context

Context documents give future humans and agents the background needed to reason correctly.

Use this directory for product background, technical background, confirmed facts, assumptions, related wiki pages, related workspaces, and knowledge gaps.

Create context documents with `./scripts/scaffold.sh create-doc <workspace-name> context <topic-slug>`. Use [../../../templates/documents/context.md](../../../templates/documents/context.md) manually only when the CLI cannot express the specialized file.
